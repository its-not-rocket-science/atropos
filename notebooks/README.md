# Atropos Notebooks

Interactive examples demonstrating Atropos usage.

## Getting Started

1. Install Jupyter and dependencies:
```bash
pip install jupyter pandas matplotlib
```

2. Install Atropos in editable mode:
```bash
cd ..
pip install -e .
```

3. Launch Jupyter:
```bash
jupyter notebook
```

## Notebooks

| Notebook | Description |
|----------|-------------|
| `01_getting_started.ipynb` | Basic usage: scenarios, strategies, and interpretation |
| `02_batch_analysis.ipynb` | Batch processing and comparison across multiple scenarios |
| `03_custom_strategies.ipynb` | Defining and optimizing custom strategies |

## Output

Notebooks may save reports to `../reports/` directory.
